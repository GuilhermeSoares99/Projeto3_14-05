## ⚡ Entrega do Dia 14/05/2026: Interatividade Avançada com JavaScript

Nesta etapa de consolidação, unificamos a estrutura semântica (Aula 30/04) e as regras de design responsivo (Aula 07/05), injetando comportamentos dinâmicos por meio da manipulação do DOM conforme as Diretrizes da Missão.

### Recursos Implementados (Atendimento aos Requisitos):

1. **Manipulação Limpa do DOM:** Seleção rigorosa de elementos utilizando `document.getElementById` no arquivo `script.js`, mantendo o código organizado e sem misturar JavaScript dentro das tags HTML.
2. **Tríade Completa de Eventos Exigidos:**
   * **Evento de Clique (`click`):** Aplicado nos botões de alternância. O clique altera o texto descritivo e injeta classes dinâmicas de estilo CSS no bloco do conteúdo.
   * **Evento de Mouse (`mouseover` / `mouseout`):** Captura a ação do usuário ao passar o ponteiro sobre a imagem, aplicando um efeito visual de zoom e brilho fluido.
   * **Evento de Teclado (`keyup`):** Monitora em tempo real a digitação do usuário dentro do formulário, gerando um feedback de eco imediato e contando os caracteres.
3. **Reset de Estados:** Um botão dedicado a reverter todas as manipulações de estilo e limpar os dados inseridos nos campos, devolvendo a integridade inicial da página.
